# Copy Tab URLs

This is a Chrome extension that allows to you copy multiple open URLs at once.
Ever had a bunch of tabs you had open and wanted to send to someone else? Or save in your own notes? Now you can save some time copying each link one-by-one, and instead just hit a button to copy all the URLs to your clipboard.
